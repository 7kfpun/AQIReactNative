## Running

#### Clone & install

* Clone this repo `git clone git@github.com:7kfpun/AQIReactNative.git`
* `cd AQIReactNative`
* run `npm install` || run `yarn install`

#### iOS

* Run `react-native run-ios`

#### Android

* Run `android avd` and start an emulator
* Run `react-native run-android`
